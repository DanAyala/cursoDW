ARCHIVOS DE PRACTICAS REALIZADAS PARA EL CURSO DE DESARROLLO WEB

En el curso antes mencionado se utilizarán herramientas para desarrollar páginas web.
Las herramientas que se enseñaran son las siguientes:
•	HTML5
•	CSS3
•	JAVASCRIPT
•	JQUERY
•	PHP
•	MYSQL
El curso consiste en explicar cada una de sus herramientas, comenzando por el Front-end y terminando con el back-end. Se realizarán practicas a lo largo del curso para explicar y practicar cada una de las herramientas y se realizará un proyecto aplicando todas ellas.
